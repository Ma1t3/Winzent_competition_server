PG-ASC Competition Server

Welcome to Agent System Competition Server. It is developed by the project group "Agent System Competition" (PG-ASC) at the University of Oldenburg. The purpose of this server is to provide a platform for evaluating the performance of agent systems for managing a power grid in experiments and competitions. The implementation of the server is build on top of the palaestrAI framework. To provide a realistic power grid simulation, the server uses the MIDAS framework which is build on the Smart Grid co-simulation framework mosaik. If you are new to the server, please read the How-To-Page first. You can see public agent systems, experiments and competitions without logging in. To upload your own agent systems and use them in experiments or to compare them in competitions you need to create an account.
